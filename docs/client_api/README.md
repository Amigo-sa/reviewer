В этой папке находятся файлы коллекций экспортированных из программы postman в формате JSON

Для импорта файла надо произвести следующие действия:
1) Установить postman https://www.getpostman.com
2) Выбрать File->Import->Choose Files

Файл с client api для просмотра из браузера опубликован по ссылке:
https://documenter.getpostman.com/view/4747165/RWaDWBCi
